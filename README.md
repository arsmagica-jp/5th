# アルスマギカ 第5版

アルスマギカ第5版の日本語訳のリポジトリです。

2010年代に有志で翻訳していたものを掲載しています。
当時のメンバーの中に連絡の取れない方がいまして、その方の担当分は除外しています。その分については、今後新たに翻訳して掲載してまいります。
翻訳・改善にご協力いただける方がいらっしゃいましたら、 [氷川](https://x.com/kilica)までご連絡ください。
